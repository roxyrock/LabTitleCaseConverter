# Title Case Converter

A small JavaScript utility that transforms any input string into proper title case (freeCodeCamp Lab).  
The function capitalizes the first letter of each word while converting the rest of the characters to lowercase.

## Features
- Converts any string to title case  
- Handles mixed‑case input  
- Simple, readable implementation  

## Example
```js
titleCase("javaScript is fun");
// → "Javascript Is Fun"
